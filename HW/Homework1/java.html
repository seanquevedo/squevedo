Java





Like C++, Java is an Object-Oriented Programming (OOP) language. Java is known for it's garbage collection - this feature destroys objects that are no longer in use when running a program. Java is also known for using single class, hierarchical inheritance. Although Java's speed is slow, it is very efficient to program for rigorous applications.


Java's syntax to declare or print variables are not very easy to follow, but the syntax to create methods and classes are easy to follow and organized.
